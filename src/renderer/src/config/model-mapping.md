## 📏 宽高比对应的实际尺寸

### 图像尺寸 (来自 constants.go)
- `1:1` → 1024x1024
- `3:4` → 864x1152
- `4:3` → 1152x864
- `16:9` → 1280x720
- `9:16` → 720x1280
- `2:3` → 832x1248
- `3:2` → 1248x832
- `21:9` → 1512x648

### 视频宽高比
- `16:9` → 1280×720 (默认)
- `9:16` → 720×1280
- `1:1` → 960×960
- `4:3` → 960×720
- `3:4` → 720×960
- `21:9` → 1680×720
